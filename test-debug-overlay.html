Debug Overlay Test Page

Instructions:

  • This page has been created to test the debug overlay functionality
  • The actual debug overlay is integrated into the React app at /src/web/App.tsx
  • To test the debug overlay:
    1. Run npm run dev in your terminal to start the development server
    2. Open http://localhost:5173 in your browser
    3. Press Ctrl+Shift+D (or Cmd+Shift+D on Mac) to toggle the debug overlay
  • Test on different devices:
    • Desktop: Resize browser window to see measurements change
    • Mobile: Use browser dev tools device simulation or actual mobile device
    • Orientation: Rotate mobile device to test orientation changes

Screen Measurements Expected:

  • Screen Width/Height: Physical screen dimensions
  • Available Width/Height: Screen minus OS taskbar/dock
  • Inner Width/Height: Browser content area
  • Outer Width/Height: Full browser window
  • Device Pixel Ratio: Display density (usually 1.0 on desktop, 2.0+ on high-DPI)

Mobile-Specific Measurements:

  • Visual Viewport: Only visible on mobile browsers
  • Scale: Pinch-zoom level
  • Offset Left/Top: Pan position when zoomed
